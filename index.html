    About Me

    Hello, I'm Siddharth Ranka, a dedicated Web Developer and Computer Science & AI student at J.K. Lakshmipat University, Jaipur, currently in an academic exchange program at IIT Jodhpur. My journey in the world of web development began with a curiosity for building functional, user-friendly websites and has evolved into a passion for creating innovative web applications. I specialize in frontend and backend web development, working with technologies like HTML, CSS, JavaScript, React, and Next.js for dynamic user interfaces. I also have experience with MySQL for database management and Node.js for backend development, which enables me to create seamless, full-stack web solutions. My projects, such as a Multi-Modal Authentication System and a Personal Finance Visualizer, reflect my love for developing secure, user-centric applications. I’ve also dabbled in Machine Learning and AI, integrating features like face recognition and data visualization to make websites not just interactive but intelligent. I’m passionate about creating web experiences that are not only functional but also engaging and accessible. When I’m not coding, you’ll find me exploring new frameworks, enhancing my skills, and contributing to the tech community.
